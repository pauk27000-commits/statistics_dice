# 🎲 Статистика бросков кубов (Dice Statistics for SWADE)

Интерактивный аналитический модуль для Foundry VTT, разработанный специально под систему **Savage Worlds Adventure Edition (SWADE)**. Он собирает детальную информацию о бросках игроков, отслеживает рекорды сессий, рассчитывает вероятность удачи и позволяет обменивать успехи на ценный игровой ресурс — фишки (Бенни)!

---

## ✨ Ключевые возможности

* 📊 **Подробная аналитика бросков**:
  * Модуль отслеживает броски каждого типа куба (d4, d6, d8, d10, d12) и дикого куба (Wild Die).
  * Разделение статистики на **текущую игровую сессию** и **за всё время**.
* 🏆 **Таблица рекордов**:
  * Автоматический расчёт и отображение выдающихся бросков (самый большой урон, самый высокий результат на одном кубе, рекорд взрывов кубов).
* 🎟️ **Магазин за очки удачи (LP)**:
  * Игроки зарабатывают очки удачи (Luck Points) за активную игру и броски.
  * Накопленные очки можно тратить на покупку фишек для себя, подарки сопартийцам или активацию сюжетных Воспоминаний (Flashbacks).
  * Гибкая настройка цен на все опции в магазине через меню модуля.
* 📈 **Визуальные графики**:
  * Наглядное распределение выпавших значений с помощью динамических графиков.
* 🎖️ **Достижения (Achievements)**:
  * Система ачивок, открывающихся при достижении особых условий (например, критические провалы, длинная череда взрывов и т.д.).
* 🔥 **Алтарь Удачи (Altar of Fortune)**:
  * Глобальный котел пожертвований! Игроки могут скидываться своими LP на общее благо для получения глобальной награды от Мастера.
  * Визуальное отображение прогресса и "Зал Славы" (топ подношений).
* ⚙️ **Гибкие настройки**:
  * Мастер может в любой момент выключить экономику (LP и магазин), скрыть Алтарь или полностью поставить учет статистики на паузу.
* 💾 **Импорт и экспорт**:
  * Перенос статистики между мирами одной кнопкой.

## 📋 Отслеживаемые рекорды

| Рекорд | Название в системе | Описание |
| :--- | :--- | :--- |
| 💥 **Самый высокий урон** | `highestDamage` | Максимальный урон, нанесённый одним броском. |
| 🎯 **Лучший бросок навыка** | `highestSkill` | Максимальное итоговое значение проверки навыка. |
| 🎲 **Лучший куб** | `biggestSingleDie` | Самый высокий чистый результат на одной грани куба. |
| 🔗 **Длинный взрыв** | `longestAceChain` | Самая длинная непрерывная цепочка "взрывов" кубов (Aces). |
| ⚡ **Максимум подъемов** | `mostRaises` | Наибольшее количество полученных подъемов за один бросок. |

## 🚀 Установка

Установить модуль в Foundry VTT можно по ссылке манифеста:

1. Откройте вкладку **Add-on Modules** в главном меню Foundry VTT.
2. Нажмите кнопку **Install Module**.
3. Вставьте следующую ссылку в поле **Manifest URL**:
   ```text
   https://raw.githubusercontent.com/pauk27000-commits/statistics_dice/main/module.json
   ```
4. Нажмите **Install** и дождитесь завершения установки.

## 🛠️ Внутреннее устройство (Архитектура)

Модуль состоит из нескольких независимых скриптов в папке `scripts/`:

*   `main.js` — точка входа, инициализация хуков Foundry VTT.
*   `storage.js` — класс `StatisticsStorage` для сохранения/загрузки данных из настроек мира (`dice-stats-v3` и `module-config-v1`).
*   `tracker.js` — парсер чата, который перехватывает броски кубов и считает их результаты.
*   `ui.js` / `settings-ui.js` — экран отображения статистики, рекордов и интерфейс управления для ГМа.
*   `chart.js` — отрисовка графиков распределения.
*   `achievements.js` — проверка условий и выдача ачивок игрокам.
